The Orbital Energy Node: SBSP and Distributed Grid Architecture

Infrastructure Strategy

If space-based solar power were deployed at infrastructure scale, it would not replace existing grid architecture — it would introduce a new class of input to energy grids that is locationally independent, continuously available, and transmissible to receiving stations outside conventional grid corridors. This brief examines the grid integration question: what receiving infrastructure is required, how ground rectenna fields connect to national grids, what the power delivery profile looks like, and where SBSP's grid integration characteristics differ from conventional generation sources.

Strategic Thesis

SBSP's grid integration characteristics — continuous availability, location-flexible delivery, and independence from weather and fuel supply chains — represent a qualitatively different type of grid input rather than a marginal improvement over terrestrial renewables. This does not mean deployment is imminent or economically competitive; it means that if SBSP is eventually deployed, its grid role would be structurally different from wind and solar, closer to dispatchable baseload than to variable renewable generation.

Institutional Context

Grid integration is treated in feasibility studies as a separate design layer from the orbital system. Frazer-Nash's UK study and ESA's SOLARIS assessment both examine ground segment requirements including rectenna field size, land use, grid connection infrastructure, and regulatory frameworks for a new category of transmission. These studies identify grid integration as technically tractable but requiring regulatory development, land allocation, and grid operator engagement that does not currently exist for SBSP.

Technical Context

A rectenna field receiving microwave power from an SBSP satellite would convert microwave energy directly to DC electricity, which is then conditioned for grid injection. The receiving field footprint is large — estimates from feasibility studies suggest several square kilometres for a multi-gigawatt link — but the field can be located anywhere with line of sight to the satellite's transmission footprint, including outside existing grid corridors. Power conversion efficiency across the full transmission chain (DC to microwave to DC) is a key technical constraint currently under development.

Feasibility Boundary

No SBSP-connected rectenna field has been built at grid scale. Power conversion efficiency and transmission chain losses remain subjects of active research. Grid integration feasibility is contingent on orbital system availability, which itself is not established. The grid integration analysis in feasibility studies is modelled from component performance data, not operational system data.

Claim Boundary

This brief examines the technical and regulatory characteristics of SBSP grid integration as documented in feasibility studies. It does not claim that SBSP is grid-ready, that rectenna fields are commercially feasible, or that any grid operator has committed to SBSP integration. All performance figures are from modelling studies and carry the assumptions of those studies.
