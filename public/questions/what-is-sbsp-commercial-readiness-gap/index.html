What would a commercially ready SBSP system require that does not yet exist?

Reference Question

A commercially ready SBSP system would require several capabilities that do not currently exist at the necessary scale. First, launch costs would need to fall to a level that makes the mass-to-orbit burden economically viable — current analyses suggest costs of $1,000 per kilogram or below to geostationary orbit as a threshold, which has not been achieved. Second, in-orbit assembly of very large structures (solar arrays covering multiple square kilometres) would require robotic and autonomous systems beyond current operational capability. Third, the wireless power transmission system would need to demonstrate sustained, controlled delivery at gigawatt-class power levels over 36,000 kilometres — far beyond any demonstrated experiment. Fourth, a terrestrial rectenna field covering tens of square kilometres would need to be sited, permitted, and built. Fifth, the regulatory framework for operating a space-based power transmitter in an occupied spectrum band would need to be established. No single one of these requirements has been met at the required scale.

Answer Boundary

The gap analysis is based on engineering assessments and institutional studies (ESA Solaris, JAXA SSPS, NASA SSPP). None of the threshold requirements have been met at commercial scale. The gap analysis itself is contested — different studies use different cost and performance assumptions.
