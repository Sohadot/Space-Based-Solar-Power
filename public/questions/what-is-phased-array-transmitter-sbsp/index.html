What is a phased array transmitter in the context of SBSP?

Reference Question

A phased array transmitter is an antenna system in which many individual antenna elements emit radio waves with carefully controlled timing differences — the phase — so that the emitted waves constructively interfere in a chosen direction and cancel in others. In SBSP, the transmitting satellite would carry a phased array of millions of antenna elements arranged across a large panel, producing a focused microwave beam directed at the ground rectenna. By adjusting the phase of each element electronically, the beam can be steered without mechanically moving the antenna — a capability called electronic beam steering. This is essential because even in geostationary orbit, pointing corrections are continuously needed to keep the beam on the small terrestrial target. The phased array approach has been demonstrated in radar and communications systems on Earth. For SBSP, the challenge is operating a phased array at the power levels, aperture sizes, and efficiencies required for gigawatt-class power delivery — which has not been demonstrated at commercial scale.

Answer Boundary

Phased array technology is mature in radar, communications, and small-scale power beaming experiments. SBSP-scale phased arrays for gigawatt power delivery have not been built or demonstrated.
